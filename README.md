Task-1 : Problem Statement: Student Result Processing System
This Repo contains main.c program which reads student details from an input file. 
According to the task given, the program 
 1) Validates the data
 2) Computes Total marks and grade
 3) Generates a report(output.txt)
 4) Generates error log(error.txt)

Program accepts input file name from command line(CLI).
